*포인터란?
"포인터" : 메모리 상에 위치한 특정한 데이터의 (시작)주소값을 보관하는 변수!

*포인터의 정의!
(포인터에 주소값이 저장되는 데이터의 형) *(포인터의 이름);
(포인터에 주소값이 저장되는 데이터의 형)* (포인터의 이름);

ex)
예를 들어 p 라는 포인터가 int 데이터를 가리키고 싶다고 하면
int* p;  // 라고 하거나
int* p;  // 로 하면 된다
라 하면 올바르게 됩니다. 
즉 위 포인터 p 는 int 형 데이터의 주소값을 저장하는 변수가 되는 것 입니다.

*&연산자(=AND연산자)
AND는 원래 두 개의 피연산자를 가지지만, 여기에서는 1개의 연산자만을
가진다. 이런 연산자를 '단항'연산자라고 한다.

*&정의
& (주소값을 계산할 데이터)

* *연산자(=곱 연산자)

*연산자는 포인터 들에게만 적용된다는 사실! 잊지 말자!!!!!(Don't forget!)

*& 와 *연산자의 역할은 무엇인가요?

&는 변수 이름앞에 붙어서 주소값을 불러오는 기능을 담당한다.

*는 int *p;로 쓰일때는 p를 주소값에 대입할수있는 변수로 만들어 주고,
*p로 쓰일때는 주소값에있는 변수를 나타낸다.
