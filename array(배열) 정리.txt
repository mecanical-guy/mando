*int형 배열이란?
배열의 각 원소가 "int 형의 변수"인것을 뜻한다.

*int형의 배열의 배열이란?
배열의 각 원소가 "int형의 배열"인것을 뜻한다.

*형태

(배열의 형) (배열의 이름) [?][?]
배열의 형 자리에 "int"가 오면되고, ?에는 "배열의 크기"가 온다.

★따라서 arr[m][n] 과 같이 배열을 선언한다면, m x n개의 변수를 가지는 배열을
선언하게 된다.

★컴퓨터 메모리에서는 2차원으로 만들어 지지 않고, 선형적으로 배열된다.
그렇기 때문에, 2차원으로 생각하게되면 보기도 쉽고 이해하기 간편하기 때문에
2차원으로 배열된다라고 생각하면 좋다.

ex)
1. int arr[] = {1, 2, 3, 4};
2. int arr[4] = {1, 2, 3, 4};
둘다 컴파일 된다. 왜냐? 컴퓨터는 우리가 배열을 집어 넣은것을보고 아! 이사람은
배열에 4개의 원소를 집어 넣었으니까 []안에는 '4'가 들어가겠구나라고 생각한다.

cf)
int arr[];
이 경우는 '배열의 크기가 임의로 정해지지 않았기 때문에, 컴퓨터는 우리가
어떤 크기의 배열을 정의 할것인지 알아 차릴수 없기때문에
오류가 발생할것이다.
